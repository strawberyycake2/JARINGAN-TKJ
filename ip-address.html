IP Address

Materi Lengkap dan Terperinci SMK TKJ

1. Pengertian IP Address

IP Address (Internet Protocol Address) adalah alamat unik yang dimiliki oleh setiap perangkat dalam jaringan komputer. Alamat ini digunakan sebagai identitas agar perangkat dapat saling mengenali dan berkomunikasi.

Tanpa IP Address, komputer tidak akan tahu ke mana data harus dikirim dan dari mana data diterima. Oleh karena itu, IP Address adalah komponen paling penting dalam jaringan komputer.

2. Fungsi IP Address

IP Address memiliki beberapa fungsi utama, yaitu:

  • Sebagai identitas perangkat dalam jaringan
  • Menentukan tujuan pengiriman data
  • Membedakan satu perangkat dengan perangkat lain
  • Memungkinkan komunikasi antar perangkat

3. Analogi IP Address dalam Kehidupan Sehari-hari

IP Address dapat dianalogikan seperti alamat rumah. Jika seseorang ingin mengirim surat, maka ia membutuhkan alamat tujuan yang jelas. Jika alamat salah atau tidak ada, surat tidak akan sampai.

Begitu juga dalam jaringan komputer. Jika IP Address salah, maka data tidak akan sampai ke perangkat tujuan.

4. Bentuk dan Contoh IP Address

IP Address yang sering digunakan saat ini adalah IPv4, yang terdiri dari 4 kelompok angka, dipisahkan oleh tanda titik (.).

Contoh IP Address:

  • 192.168.1.1
  • 192.168.1.10
  • 10.10.10.1

Setiap angka memiliki nilai dari 0 sampai 255.

5. Jenis-Jenis IP Address

a. IP Address Private

IP Address private digunakan dalam jaringan lokal (LAN) dan tidak dapat diakses langsung dari internet.

  • 192.168.x.x

b. IP Address Public

IP Address public adalah IP yang dapat diakses dari internet. Biasanya diberikan oleh ISP (penyedia layanan internet).

6. IP Address Statis dan Dinamis

a. IP Address Statis

IP Address statis adalah IP yang tidak berubah. Biasanya digunakan pada server agar alamatnya selalu tetap.

b. IP Address Dinamis

IP Address dinamis adalah IP yang bisa berubah-ubah. Biasanya diberikan otomatis oleh DHCP.

7. Kesalahan Umum dalam Penggunaan IP Address

  • IP Address bentrok (sama dengan perangkat lain)
  • Subnet tidak sesuai
  • Gateway salah
  • Pengaturan IP tidak sesuai jaringan

8. Contoh Soal dan Pembahasan

Soal: Apa fungsi IP Address dalam jaringan komputer?

Jawaban: IP Address berfungsi sebagai alamat identitas perangkat agar dapat saling berkomunikasi dan bertukar data.

9. Kesimpulan Akhir

IP Address adalah komponen utama dalam jaringan komputer. Tanpa IP Address, komunikasi antar perangkat tidak dapat dilakukan.
