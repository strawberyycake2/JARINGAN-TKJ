Server Jaringan

Materi Lengkap dan Terperinci untuk SMK TKJ

1. Pengertian Server

Server adalah sebuah komputer atau sistem yang berfungsi untuk menyediakan layanan kepada komputer lain yang disebut client di dalam sebuah jaringan.

Server bekerja dengan cara menerima permintaan (request) dari client, kemudian memproses permintaan tersebut dan mengirimkan kembali hasilnya kepada client.

Tanpa server, jaringan komputer tidak dapat berjalan secara maksimal, terutama pada jaringan skala besar seperti sekolah, kantor, atau internet.

2. Fungsi Server dalam Jaringan

  • Menyimpan dan mengelola data
  • Mengatur lalu lintas jaringan
  • Melayani permintaan client
  • Menjaga keamanan jaringan
  • Mengatur hak akses pengguna

3. Perbedaan Server dan Client

Server Client
Menyediakan layanan Menggunakan layanan
Selalu aktif Bisa aktif / nonaktif
Spesifikasi tinggi Spesifikasi standar

4. Jenis-Jenis Server

A. File Server

File server adalah server yang berfungsi untuk menyimpan dan membagikan file kepada client dalam jaringan.

Contohnya: server penyimpanan tugas siswa di sekolah.

B. Web Server

Web server adalah server yang berfungsi untuk menampilkan halaman website melalui browser seperti Chrome atau Firefox.

Contoh software web server: Apache, Nginx.

C. Database Server

Database server berfungsi untuk menyimpan dan mengelola data dalam jumlah besar.

Contoh: MySQL, PostgreSQL.

D. DNS Server

DNS server berfungsi untuk mengubah nama domain menjadi alamat IP.

E. DHCP Server

DHCP server berfungsi untuk memberikan alamat IP secara otomatis kepada client.

5. Sistem Operasi Server

Server membutuhkan sistem operasi khusus yang dirancang untuk bekerja 24 jam nonstop.

  • Linux Server (Ubuntu Server, Debian)
  • Windows Server

6. Hubungan Server dengan IP Address dan Topologi

Server wajib memiliki IP Address yang bersifat tetap (static), agar client dapat selalu terhubung ke server.

Server biasanya diletakkan di pusat topologi jaringan, terutama pada topologi star.

7. Kesalahan Umum dalam Pengelolaan Server

  • Tidak memberikan IP statis
  • Tidak melakukan backup data
  • Tidak mengamankan server
  • Server sering mati

8. Contoh Soal

Soal: Mengapa server harus menggunakan IP statis?

Jawaban: Agar client selalu dapat terhubung ke server tanpa perubahan alamat IP.

9. Kesimpulan Akhir

Server merupakan pusat layanan dalam jaringan komputer. Semua client bergantung pada server untuk mendapatkan layanan.

Pemahaman server sangat penting sebelum mempelajari web hosting, database, dan jaringan internet.
